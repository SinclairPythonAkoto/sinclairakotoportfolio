Jobs Log

This was created for a member of the Audio Visual department within a corperate establishment. After each job they complete, it is recored in their spreadsheet on their laptop. The client asked for a way to record a job, but without the need to access their laptop.

The parameters are alraedy set, so all the user needs to do is enter the data & press enter; once that happens the information is stored in a (PostgreSQL) database, then reloaded back onto the page for the user to see. I have created a reports page where the user can run a report for the different type of jobs, and the different type of outcomes (if completed or not ect). This puts the user at a variable position when they can access & update their table at anytime, with the use of a smartphone or device. As I deepen my skillset I will apply it to this app, making it a little more intricate as I progress.

Sinclair Connect

A simple app that uses an API to send SMS messages to UK numbers. Being aable to send a message to another UK number in case of an emergency (for instance if your phone died), would be seen as an advantage of this web app.

An expansion of this web app will be created when I come more comfortable with using different types of API's within apps. I will create extra functions like how to send emails and messages to social media through the web app.



Jonny And Friends For Autism (Contact Service)

The owner of this charity wanted to engage with families who are affected by autism (directly/indirectly) through her website and amongst other means. I sugested one of the ways to achieve this would be to create a platform where her demographic could contact her through SMS on her webpage; in return she would recieve the text and respond back to the person via her website. The reciever would then get a response via text message, whcih would appear to have been sent from her charity website - Jonny And Friends For Autism.

I have also included an email functionality page; once logged into the admin page the owner can not only send response SMS messages, but can also email any person with a valid email address. The email being sent it automated with a footer signed by the owner and their Jonny And Friends For Autism logo.

Key Features:

  • Send text messages to UK numbers
  • Recieve SMS sent from web app
  • Log in: You will be redirected to login if an attempt to access the admin page without signning in.
  • Send emails to any valid email address. Each email being sent is attached with a footer & logo at the bottom.
